---
title: "Get Missed Chats"
api: "GET https://api-v2.opencopilot.so/reports/{copilotId}/chats/missed"
description: "Retrieve the count of missed chats based on specified criteria"
---

### Path Parameters

<ParamField path="copilotId" type="string" required>
  The ID of the copilot to filter chats
</ParamField>

### Query Parameters

<ParamField query="startDate" type="string" format="date-time">
  The start date for the date range filter (ISO 8601 format)
</ParamField>

<ParamField query="endDate" type="string" format="date-time">
  The end date for the date range filter (ISO 8601 format)
</ParamField>

### Response

<ResponseField name="count" type="number">
  The number of missed chats based on the specified criteria
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api-v2.opencopilot.so/reports/abc123/chats/missed?startDate=2023-01-01T00:00:00Z&endDate=2023-12-31T23:59:59Z' \
--header 'Authorization: Bearer YOUR_TOKEN'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "count": 42
}
```

</ResponseExample>

---
title: "Get Staffing Prediction"
api: "GET https://api-v2.opencopilot.so/reports/{copilotId}/agents/staffing-prediction"
description: "Retrieve a prediction for staffing needs based on historical chat data"
---

This endpoint provides a staffing prediction based on historical chat data. It analyzes the number of chats per hour to help optimize agent allocation and improve response times.

### Path Parameters

<ParamField path="copilotId" type="string" required>
  The ID of the copilot
</ParamField>

### Query Parameters

<ParamField query="startDate" type="string" format="date-time" optional>
  The start date for the date range to analyze (ISO 8601 format)
</ParamField>

<ParamField query="endDate" type="string" format="date-time" optional>
  The end date for the date range to analyze (ISO 8601 format)
</ParamField>

### Response

<ResponseField name="hourlyChats" type="array">
  An array of objects representing the chat volume for each hour
  <Expandable title="properties">
    <ResponseField name="hour" type="number">
      The hour of the day (0-23)
    </ResponseField>
    <ResponseField name="chatCount" type="number">
      The number of chats during that hour
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api-v2.opencopilot.so/reports/abc123/agents/staffing-prediction?startDate=2023-01-01T00:00:00Z&endDate=2023-01-31T23:59:59Z' \
--header 'Authorization: Bearer YOUR_TOKEN'
```

</RequestExample>

## Overview

Our Jira integration helps you turn customer feedback into actionable tickets. Whether it's feature requests, bug reports, or general feedback, our AI can automatically create and categorize Jira tickets based on customer conversations.

## Setup Process

<Steps>
  <Step title="Connect Jira">
    Link your Jira workspace to Open.cx:
    - Go to Integrations in your dashboard
    - Select Jira
    - Authorize with your Atlassian account
  </Step>
  
  <Step title="Configure Projects">
    Set up project mapping:
    - Choose target projects
    - Define issue types
    - Set up custom fields
    - Configure workflows
  </Step>
  
  <Step title="Define Rules">
    Create ticket creation rules:
    - Priority mapping
    - Label assignments
    - Template customization
    - Notification settings
  </Step>
</Steps>

## How It Works

1. **Feedback Detection**: AI identifies actionable feedback in conversations
2. **Smart Analysis**: Categorizes the type of feedback and required action
3. **Ticket Creation**: Automatically creates properly formatted Jira tickets
4. **Follow-up**: Keeps customers informed about ticket status

## Zendesk Integration

Transform your Zendesk support with AI-powered assistance. Our integration enables:
- Automatic ticket handling
- Knowledge base access
- Smart ticket routing
- Historical context awareness

### Setup Process

<Steps>
  <Step title="Connect Zendesk">
    Link your Zendesk account from the Open.cx dashboard
  </Step>
  <Step title="Configure Access">
    Set up API credentials and permissions
  </Step>
  <Step title="Select Data">
    Choose what to sync:
    - Support tickets
    - Help center articles
    - User conversations
    - Macros and templates
  </Step>
</Steps>

### Features
- Automatic ticket resolution

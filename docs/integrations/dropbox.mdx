## Dropbox Integration

Use your Dropbox as an AI knowledge source:
- Access all file types
- Process shared folders
- Automatic updates
- Smart content extraction

### Setup Process

<Steps>
  <Step title="Connect Dropbox">
    Link your Dropbox account to Open.cx
  </Step>
  <Step title="Select Files">
    Choose what to include:
    - Documents
    - Shared folders
    - Team spaces
    - File types
  </Step>
  <Step title="Configure Sync">
    Set up sync preferences:
    - Update frequency
    - File filters
    - Access rules
  </Step>
</Steps>

### Features
- File parsing
- Auto-sync
- Version tracking
- Permission management
